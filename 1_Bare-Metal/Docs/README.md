# Hardware Documentation & Reference Data

This directory contains the immutable silicon blueprints and reference materials used to architect the bare-metal drivers.

* **RM0090 Reference Manual:** The primary text for memory mapping, bus architecture, and peripheral register configuration.
* **STM32F407VG Datasheet:** Used for electrical characteristics, pin multiplexing (Alternate Functions), and physical packaging.
* **Cortex-M4 Programming Manual:** Used for core CPU registers, NVIC (Interrupts), and SysTick configuration.
* **STM32F4 Discovery User Manual:** Used for board-level schematics (LEDs, Pushbuttons, Audio DAC).
